## Matéria inicial
#### DER (Diagrama Entidade-Relacionamento)
O DER (Diagrama Entidade-Relacionamento) é uma ferramenta essencial no design de bancos de dados. Ele ajuda a visualizar e entender como as diferentes entidades de um sistema se conectam entre si. Basicamente, é um mapa que facilita a organização dos dados, mostrando relações e dependências. Ao utilizar o DER, podemos planejar de forma mais eficiente e evitar problemas no futuro, garantindo que o banco de dados seja bem estruturado e funcione de maneira otimizada.
